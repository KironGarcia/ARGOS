====== PAM8403 Audio Amplifier Module ======
Created Monday 10 February 2026

===== Overview =====

The PAM8403 is a Class-D stereo audio amplifier module that amplifies the weak DAC output from ESP32 to drive a speaker. It provides clear, efficient audio output for the "Alexa China" voice responses.

**Role in Project:** Amplify ESP32 DAC signal to drive 4Ω 3W speaker  
**Type:** Class-D amplifier (high efficiency)  
**Connection:** ESP32 DAC output → PAM8403 → Speaker  
**Power:** 3.3V or 5V (depending on module version)

---

===== Technical Specifications =====

==== General Characteristics ====

* **Type:** Class-D stereo audio amplifier
* **Output Power:**
  - 3W per channel @ 4Ω load, 5V supply
  - 2W per channel @ 4Ω load, 3.3V supply
* **Efficiency:** 90% (typical) - very efficient
* **Total Harmonic Distortion (THD):** <1% @ 1W output
* **Signal-to-Noise Ratio (SNR):** >70 dB

* **Recommended Voltage:** 5V (for maximum power)

  - 3W output: ~300-400 mA (peak)
* **Input Impedance:** 20 kΩ (typical)
* **Output Impedance:** Matched to speaker (4Ω or 8Ω)

==== Physical Specifications ====

* **Module Size:** ~20 mm × 15 mm (typical breakout board)

* **Operating Temperature:** -40°C to +85°C

===== Pin Configuration =====

==== PAM8403 Module Pinout ====

**Typical Module Layout:**
```
PAM8403 Module
├── VDD  → Power supply (2.5V - 5.5V)
├── GND  → Ground
├── IN+  → Audio input (positive)
├── IN-  → Audio input (negative/GND)
├── OUT+ → Speaker output (positive)
├── OUT- → Speaker output (negative)
└── SD   → Shutdown control (optional)
```

**Pin Functions:**
* **VDD:** Power supply input (2.5V - 5.5V)
  - 5V recommended for maximum power
  - 3.3V works but lower output
* **GND:** Ground reference
* **IN+:** Audio input signal (positive)
  - Connects to ESP32 DAC output
  - AC-coupled (capacitor recommended)
* **IN-:** Audio input signal (negative)
  - Can connect to GND (single-ended)
  - Or use differential input
* **OUT+:** Speaker output (positive terminal)
* **OUT-:** Speaker output (negative terminal)
* **SD (Shutdown):** Enable/disable control
  - HIGH = amplifier enabled
  - LOW = amplifier disabled (low power)
  - Can be left floating (always enabled)

===== Connection to ESP32 =====

==== Wiring Diagram ====

**Power Connections:**
```
PAM8403 Module    →    Power Source
├── VDD          →    5V or 3.3V (check module)
└── GND          →    ESP32 GND (common ground)
```

**Audio Input Connections:**
```
ESP32             →    PAM8403 Module
├── GPIO 25      →    IN+ (DAC1 output)
└── GND          →    IN- (or connect to GND)
```

**Audio Output Connections:**
```
PAM8403 Module    →    Speaker
├── OUT+         →    Speaker positive terminal
└── OUT-         →    Speaker negative terminal
```

**Complete Connection:**
```
ESP32 GPIO 25 (DAC1) → PAM8403 IN+
ESP32 GND            → PAM8403 IN- (and GND)
PAM8403 VDD          → 5V or 3.3V (module dependent)
PAM8403 GND          → ESP32 GND
PAM8403 OUT+         → Speaker +
PAM8403 OUT-         → Speaker -
```

===== Power Supply Considerations =====

==== Voltage Selection ====

**5V Power Supply:**
* **Advantages:**
  - Maximum output power (3W)
  - Better audio quality at high volume
  - More headroom for peaks
* **Disadvantages:**
  - Requires 5V source (ESP32 VIN is 4.8V)
  - May need separate 5V regulator

**3.3V Power Supply:**
* **Advantages:**
  - Can use ESP32 3.3V output
  - Simpler wiring
  - Lower power consumption
* **Disadvantages:**
  - Lower output power (~2W)
  - Less headroom for loud audio

**Recommendation:**
* Use 5V if available (better performance)
* Use 3.3V if simplicity is priority
* Check module specifications (some modules require 5V)

===== Audio Signal Path =====

==== Signal Flow ====

**ESP32 DAC → PAM8403 → Speaker**

**Step 1: ESP32 DAC Output**
* GPIO 25 (DAC1) outputs analog voltage
* Range: 0V - 3.3V
* Resolution: 8-bit (256 levels)
* Sample rate: 8-16 kHz (for TTS)

**Step 2: PAM8403 Amplification**
* Input: 0-3.3V analog signal
* Amplification: ~10-20x (gain depends on module)
* Output: 0-5V (or 0-3.3V) PWM signal
* Class-D modulation (efficient switching)

**Step 3: Speaker Output**
* PWM signal filtered by speaker inductance
* Converts to analog audio waves
* 4Ω impedance matches amplifier
* 3W power output

===== Class-D Amplifier Operation =====

==== How Class-D Works ====

**Class-D amplifiers use PWM (Pulse Width Modulation) to amplify audio efficiently.**

**Process:**
1. **Input:** Analog audio signal (0-3.3V)
2. **Modulation:** Convert to PWM signal (high frequency switching)
3. **Amplification:** Switch power transistors ON/OFF
4. **Filtering:** Speaker inductance filters PWM to analog
5. **Output:** Clean audio waves to speaker

**Advantages:**
* ✅ High efficiency (90% vs 50% for Class-AB)
* ✅ Low heat generation
* ✅ Small size
* ✅ Low power consumption

**For ARGOS:**
* Perfect for battery-powered system
* Low heat = safe in car enclosure
* Efficient = longer battery life

===== Troubleshooting Guide =====

==== Common Issues ====

**Problem: No audio output**
* Check: Power supply (VDD connected)
* Check: GND connection (common ground)
* Check: Input signal (ESP32 DAC output)
* Check: Speaker connection (OUT+ and OUT-)
* Check: Speaker impedance (should be 4Ω or 8Ω)

**Problem: Distorted audio**
* Check: Power supply voltage (should be stable)
* Check: Input signal level (may be too high)
* Check: Speaker impedance (mismatch)
* Check: Power supply current capacity

**Problem: Low volume**
* Check: Power supply voltage (5V better than 3.3V)
* Check: Input signal level (DAC output)
* Check: Speaker impedance (4Ω better than 8Ω)
* Check: Gain settings (if module has gain control)

**Problem: Buzzing/humming**
* Check: GND connection (common ground essential)
* Check: Power supply quality (clean voltage)
* Check: Cable routing (away from digital signals)
* Check: Electromagnetic interference (motors, WiFi)
