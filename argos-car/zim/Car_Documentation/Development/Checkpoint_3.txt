==== 2. Simple Menu Script Created ====

**Script:** `menu_demo.py`  
**Location (in project):** `argos-car/argos-architecture/src/menu_demo.py`

**What it does:**
* Menu-driven demo (no continuous sequence).
* Options: **1** Adelante, **2** Atrás, **3** Derecha, **4** Esquerda, **5** Salir.
* After each action the script stops and returns to the menu.
* Uses current pinout: ENA = GPIO 26 (Pin 37), ENB = GPIO 20 (Pin 38), IN1–IN4 as in Checkpoint 1.1.
* Differential drive for turns: same logic as wheel_tests.py.

**How to use it:**

**Copy script from PC to Raspberry Pi (run on your PC):**
{{{code: lang="text" linenumbers="True"
scp /home/kiron/Cursor/projeto-argos/ARGOS/argos-car/argos-architecture/src/menu_demo.py pi@argos.local:~/
}}}

**Run on the Raspberry Pi (after SSH: ssh pi@argos.local):**
{{{code: lang="text" linenumbers="True"
sudo python3 ~/menu_demo.py
}}}

**In the menu:** type 1–5 and Enter. Option 5 or Ctrl+C exits.
