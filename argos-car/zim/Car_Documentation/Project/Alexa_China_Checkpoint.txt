===== 🎤 Voice System: Simple But Effective =====

==== Wake Word + Command Matching ====

**No need for:**
* ❌ Cloud-based voice recognition
* ❌ Natural language processing
* ❌ Complex speech-to-text engines
* ❌ Machine learning models

**What we actually need:**
```python
wake_words = ["hey argos", "argos"]
commands = {
    "move forward": lambda: car.forward(),
    "stop": lambda: car.stop(),
    "turn left": lambda: car.turn_left(),
    "turn right": lambda: car.turn_right(),
    "i'm under attack": lambda: argos.simulate_attack(),
    "status": lambda: argos.report_status()
}

# Simple pattern matching, no magic needed
if wake_word_detected():
    command = recognize_command()
    execute(commands[command])
    speak_response("Command executed")
```

**TTS (Text-to-Speech) Responses:**
* `espeak` (lightweight, works offline)
* `pico2wave` (better quality, still offline)
* Pre-recorded audio clips (fastest, most reliable)

**This is enough for a KILLER demo.**
